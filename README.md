# int3_1regulars

## Для запуска:

1. Склонируйте репу
   `git clone https://github.com/artias13/int3_1regulars`
2. Соберите venv
   `python -m venv venv`
3. Активируйте
   `.\.venv\Scripts\activate`
4. Установите зависимости
   `pip3 install -r .\requirements.txt`
5. Запустите тест
   `pytest .\test.py`
